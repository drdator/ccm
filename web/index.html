Getting Started

1

Install CCM CLI

npm install -g ccm-cli

Install the CCM command-line tool globally on your system.

2

Search & Install

ccm search git
ccm install dev-tools

Find useful command packages and install them in your project.

3

Use in Claude Code

/dev-tools/git-helper

Access installed commands in Claude Code with package namespacing.

Publishing Your Own Commands

Terminal
# Create a new command package
$ mkdir my-commands && cd my-commands
$ ccm init --name "my-utils"
# Create commands in commands/ directory
$ echo "# Hello Command" > commands/hello.md
# Register and publish
$ ccm register
$ ccm publish
Package Metadata

Enhance your package with rich metadata in ccm.json:

ccm.json
{
"name": "my-dev-tools",
"version": "1.0.0",
"description": "Essential development utilities",
"category": "development",
"license": "MIT",
"repository": "https://github.com/user/my-dev-tools",
"homepage": "https://mydocs.com",
"keywords": ["git", "docker", "productivity"]
}
